# My custom Tada68 layout.

Base Layout
-----------
* Changed Win to Fn
* Changed Grave to Pscr
* Changes ESC to Grave

Function Layout
---------------
* Removed all except F row
* Arrow keys are PgUp, PgDn, Home, End
* W is Win
* Delete is Mute
* PgUp, PgDn are Volume Up/Down
* <,> and ? are Backlight controls
* ESC is ESC
